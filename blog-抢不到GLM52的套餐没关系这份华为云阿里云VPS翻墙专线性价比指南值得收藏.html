四、实战：从零搭建一台伪装成Web应用的翻墙VPS

以下步骤基于华为云轻量应用服务器（Ubuntu 22.04），配置时间约15分钟。

Step 1：初始化服务器

apt update && apt upgrade -y
apt install curl wget git vim -y

Step 2：安装V2Ray并配置WebSocket+TLS

使用一键脚本（如233boy的v2ray脚本），选择WebSocket+TLS模式，并填入你的域名（需提前解析到服务器IP）。注意：证书必须使用Let's Encrypt自动申请，避免自签名证书被墙。

Step 3：伪装成真实的Web应用

为了应对“严打违规跨境宽带”，你需要部署一个真实的Golang Web应用（比如一个简单的博客）。可以写一个返回空值的/health接口，或者部署一个静态页面。这样即使被运营商探测流量，也会被识别为正常的HTTP请求。

Step 4：测试与优化

  • 使用speedtest-cli测试下行速度。
  • 使用mtr检查延迟和丢包率。
  • 如果发现QoS，尝试切换端口（如使用8080或8443）。
