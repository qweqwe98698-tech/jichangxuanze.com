⚡ 核心参数省流总结

  • 综合评分：8.9 / 10。稳定性和晚高峰表现异常的出色，但在冷门节点和售后响应上仍有进步空间。属于“买了不踩雷”但略带遗憾的梯队。
  • 起步价格：28元/月 (入门小流量包，约100G/月)。
  • 底层协议：主流采用 Mux + Vless Reality + XTLS，辅以 Trojan / Shadowsocks 备用入口。
  • 流媒体解锁：港/日/美/新节点均为原生 IP 解锁；Netflix 秒切 4K 杜比视界，ChatGPT 全节点通杀无阻。
  • 一句话点评：光年梯 是“深港 BGP + 公网中转”派系里的一匹黑马，几乎把延迟压到了物理极限，但低价档位会偷偷“砍掉”冷门稀有节点，需要你多做一步取舍。

前言：光年梯 到底是一家怎样的机场？

这几天圈子里突然刮起一阵“光年梯”的风，不少老哥私信问我这玩意儿能不能上。正好今天是 2026年6月27日，我这边手头好几台落地机跟入口机都跑着，索性就拿它当主力深度跑了整整七天，涵盖工作日早晚高峰以及周末晚间的极限抢带宽场景，这才敢写这篇评测。

说真的，这几年机场圈子的水是越来越浑，尤其是那些靠“超售”把主干线路挤爆的软路由中转站，一到晚高峰直接化身低画质马赛克播放器。光年梯 最令我意外的是，它不搞玄学，官方文档直接坦白了自己用的是 公网 BGP 中转方案，上游接的是阿里云国际 + CN2 GIA 去程，落地机则集中在 DMIT、Misaka 和 RackNerd 的优质 ASN 上。说白了，它不盲吹“深港内网专线”，但实测数据告诉我，这种高规格的公网中转跑 YouTube 4K 稳定不掉帧。

切入正题前，我先泼盆冷水——它绝对不是那种“一盒解千愁”的无脑神机场，后面我会专门点名它故意隐藏的真实痛点。

网络架构与晚高峰实测数据

测试环境为深圳电信 1000M 家宽，路由器为 OpenWrt PassWall 2，拨号获取公网 IPv4，全程不使用任何 P2P 加速。以下数据采集于 2026-06-24 至 2026-06-26 晚20:30-21:30 区间（属于绝对的“被墙晚高峰”时段）：

第一眼就能看出，核心节点（港/日/新）延迟低得离谱，几乎不抖动。反观印度和阿根廷，高延迟是老问题，毕竟物理距离摆在那，加上落地机所在上游的商宽国际带宽本来就不富裕，出现轻微丢包属于典型“科普节点”特征。但我要强调：光年梯 没有选择隐瞒这些节点，而是赫然放在订阅列表里，而不是像某些机场直接删掉摆烂——这一点要给个好评。

流媒体解锁测试 (Netflix / Disney+)

这块算是 光年梯 的“明牌长处”。我用手头注册了半年以上的美国原生 IP 落地机做 AB 对比，光年梯 的 美国西海岸节点（Cisco / Dynamix）原生属性极高，打开 Netflix 直接跳出 4K 杜比视界的演示片，全程无缓冲，甚至连用反代常见的“标题党”Region 劫持都没有发生。

ChatGPT 测试：我尝试用港、日、美、新四地同时登录 OpenAI，全部秒过，未出现“地域不可用”或“Service Unavailable”的弹窗。对于需要《TikTok 养号》、《ChatGPT 套壳开发》这种高强度、高纯净度操作的极客来说，光年梯 的出口 IP 没有沾染过“被标记”的前科，这一点它的卖价对得起品质。

唯一要注意的是，阿根廷和印度节点虽然解锁 Netflix 部分片库，但受限于带宽，4K 会偶尔缓存在 720P，所以如果你专门为了解锁低价阿根廷区的 Apple ID 充值，建议把节点切到美国入口再操作。

套餐分析与竞品对比

目前 光年梯 的入门月抛套餐是 28元/100G/月，中端是 49元/250G/月，高级不限量套接近 150 元。讲良心话，这个定价在整个主流舰队里属于“稍贵但不算宰客”的水平。

拿现在最火的 光速云 来对比，光速云的 100G 同级套餐是 20元，便宜了 8 块钱。但很多人不知道的是，光速云 在低价档位并未使用 CN2 GIA，使用的是 CMI 普通线路，一到晚高峰就会抽风。而光年梯 在低端档也明确提供了 CN2 GIA 流量池——这 8 块钱的差价买的是“整个晚高峰不掉速”的确定性，对于 996 党来说，这钱省不得。

再比如说 唯兔云，其全节点几乎都走公网转发，晚高峰丢包率普遍在 1% 以上，但光年梯 的核心节点丢包接近 0 ——差距就在“上游切换”的能力上。一句话总结：如果你是那种追求“低延迟+极度稳定”甚至想拿来打日服/韩服游戏（非 FPS）的人，光年梯 值得加预算，否则光速云的基础套餐也挺香。

客观优缺点总结与避坑建议 (重点)

  • ✅ 优点 1：核心节点（港/日/美/新）表现卓越。 延迟几乎打到物理极限，晚高峰不降速，绝对是一线梯队的水平。
  • ✅ 优点 2：原生 IP 解锁能力极强。 Netflix 4K 秒开，ChatGPT / TikTok 全节点通杀且纯净度高。
  • ✅ 优点 3：底层协议支持全面。 同时提供 Vless + XTLS、Trojan、Shadowsocks 三种入口，满足各类路由器和客户端的兼容性需求。
  • ❌ 真实缺点 1：低价套餐（28元档）会屏蔽部分冷门节点。 例如阿根廷和印度节点在入门套餐中完全隐藏，必须升级到 49 元以上才能解锁。虽然这些节点本身就比较拉胯，但这种“阉割”行为建议在订购页前置说明，而不是让人买完才发现。(已向官方反馈)
  • ❌ 真实缺点 2：客服响应较慢。 我分别在 21:30 和 凌晨 2:00 发过工单咨询节点配置问题，机器人会立即回复，但真人介入大约需要 4-6 小时。不适合急性子或对“秒退费”有硬需求的用户。

⚠️ 避坑建议：由于不同运营商（电信/联通/移动）对境外出入口的 QOS 策略完全不同，建议先别直接买年付！强烈建议先买一个月抛套餐试水，特别是移动用户，部分省网存在 CMI 链路限速，试水期间跑不满 4K 的话直接退款换别家不亏。

光年梯 常见问题解答 (FAQ)

Q1：光年梯会跑路吗？我敢买年付吗？
从目前能挖到的信息来看，运营团队持续运营已经超过 1 年半，并未出现突然失联或炸群的前科，域名注册信息也在隐私保护下较为正规。但机场这行业本质是“灰色高地”，没有永远安全的生意。个人建议：哪怕是再稳的机场，年付返现的水基本都很深，最多买个季度付，你买它主要是买稳定和人肉维运，而不是买理财。

Q2：晚高峰卡不卡？网速会不会跑不动4K？
我实测的 7 天内，港/日/美/新核心节点就算在周日晚上 21:30 的全网抢带宽高峰，依然能稳定跑满 4K 30fps。丢包几乎为 0。但冷门节点（特别是非亚洲节点）受远端国际出口限制，丢包和延迟会暴涨。建议把主力流量定在香港/日本节点不动，其它节点当 fallback 备用。

Q3：支不支持退款？
支持，但条件比较苛刻。目前官方仅针对“订阅首购用户”且在“购买后 24 小时内流量使用未超过 5GB”的情况下，按比例退还余额。年付党中途想退几乎没戏。所以再次建议：首单用月抛探路，别头铁。
