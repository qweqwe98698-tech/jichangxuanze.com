🖥️ Windows 电脑翻墙：Clash Verge Rev 零基础入门教程

更新时间：2026-06-06 阅读时间：约 2 分钟 适用设备：Windows 10 / 11

⚠️ 注意：本教程需要搭配稳定高速的节点使用。为了保证您的顺利配置，本教程全程使用 【唯兔云】 作为演示。如果您还没有节点，请先注册购买。

前言：为什么要用 Clash Verge Rev？

许多老玩家还在使用 Clash for Windows（俗称 CFW小猫咪）。但注意，CFW 的原作者已经删库跑路不再更新了，软件也爆出了严重的安全漏洞。现在 Windows 端公认最强、最好用的替代品就是 Clash Verge Rev，它基于更先进的 Meta 内核，自带中文，支持无脑一键导入。

第一步：下载并安装软件

1. 前往 GitHub 下载官方最新版客户端安装包：Clash Verge Rev 下载地址。

2. 下载以 .exe 结尾的安装包（通常是 x64 架构）。

3. 双击安装，一路点击“下一步（Next）”直至完成。

第二步：一键导入机场节点

不要去研究那些复杂的配置文件，直接使用全自动的一键导入功能：

1. 打开您购买的 唯兔云官网，登录您的账号进入控制台。

2. 在用户中心找到 【一键订阅】，选择 【导入到 Clash】 按钮。

3. 浏览器会提示是否打开 Clash Verge，点击“允许”。

4. 软件会自动打开并弹出一个下载进度条，进度条走完后，在软件的 “配置 (Profiles)” 页面就能看到您的机场节点列表了。请确保选中该列表（变蓝表示激活）。

第三步：一键开启科学上网

节点准备就绪，接下来我们让电脑流量接管：

1. 点击软件左侧菜单栏的 “代理 (Proxies)”，这里会展示所有可用的国家节点。我们推荐选择 【规则模式 (Rule)】，然后随便点选一个香港或日本的节点。

2. 点击左侧菜单栏的 “设置 (Settings)”。

3. 找到 【系统代理 (System Proxy)】 的开关，将其 打开。

恭喜您，配置完成！🎉

当系统代理开关变蓝后，电脑右下角任务栏的图标也会变色。现在打开您的浏览器，输入 youtube.com 测试一下吧！唯兔云的优质线路可以轻松跑满您的千兆宽带，体验极其丝滑。
